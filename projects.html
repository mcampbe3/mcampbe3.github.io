Unified Data Library Project

The Unified Data Library (UDL) is a cloud-based platform developed with the U.S. Space Force and Bluestaq to centralize and securely share space domain awareness data. It aggregates information from military, commercial, and allied sources, enabling real-time access and analysis across different security levels. The UDL supports global users and is a key component to improving defense initiatives and space operations. I work to deploy the UDL in an external environment and support the automation and advancement of the UDL.

Future of Operationally Resilient Ground Evolution Project

The Future Operationally Resilient Ground Evolution (FORGE) Project is a cloud-based, modular ground system for the U.S. Space Force. It processes data from missile warning satellites like SBIRS and Next Gen OPIR, enabling faster, more resilient threat detection. Built with an open architecture and DevSecOps approach, it supports rapid updates and potential civilian applications like disaster monitoring. I support the deployment of continuous releases of the FORGE mission data processing application framework on various environments to demonstrate the system's deployability in external conditions.
